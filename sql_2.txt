[기존]
SELECT B.CATEGORYNAME, SUM(A.QUANTITY)
FROM 
(SELECT PRODUCTID , ODD.QUANTITY FROM ORDERS OD LEFT JOIN ORDERDETAILS ODD ON OD.ORDERID = ODD.ORDERID WHERE SUBSTR(ORDERDATE,0,5)= '1996' ) A,
(SELECT CG.CATEGORYNAME, PD.PRODUCTID FROM CATEGORIES CG LEFT JOIN PRODUCTS PD ON CG.CATEGORYID = PD.CATEGORYID ) B
WHERE A.PRODUCTID = B.PRODUCTID
GROUP BY B.CATEGORYNAME

[잘못된점]
ANSI SQL 사용되었고, 함수의 잘못된 사용, 조건의 적절하지 않음. grouping 의 적절하지 않음 등이 있습니다.

[수정사항]
1.LEFT OUTER JOIN은 ANSI조인이므로 사용해서는 안됨.
2.SUBSTR(ORDERDATE,0,5)= '1996' 은 제대로된 날짜 비교가 아님.

[수정후]
SELECT B.CATEGORYNAME, SUM(A.QUANTITY) AS QUANTITY
FROM 
(SELECT PRODUCTID , ODD.QUANTITY FROM ORDERS OD , ORDERDETAILS ODD WHERE OD.ORDERID = ODD.ORDERID AND SUBSTR(ORDERDATE,0,5)= '1996' ) A,
(SELECT CG.CATEGORYNAME, PD.PRODUCTID FROM CATEGORIES CG , PRODUCTS PD WHERE CG.CATEGORYID = PD.CATEGORYID ) B
WHERE A.PRODUCTID = B.PRODUCTID
GROUP BY B.CATEGORYNAME

[부족한점]
1. LEFT조인을 오라클조인으로 바꾸면 (+)가 필요한데 이걸 넣으면 에러가 나서 제대로된 조인이 이루어져있지 않음.
2. 날짜 비교는 TO_DATE나 TO_CHAR()로 형변환후 이루어지는데 에러가 나서 잘라서 비교함. 
3. 보내주신페이지에 아래 결과와 유사하면 된다고 해서 그정도만 데이터확인. 확실한 데이터 확인은 이루어지지 않음.
